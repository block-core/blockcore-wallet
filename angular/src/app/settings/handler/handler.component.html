Register Protocol Handler

You can only register HTTP handlers from the full view. Click the expander icon at the bottom right, then open "Settings" and "Register Protocol Handler" again.

You can register this extension as handler for different protocols. This action must be done in full window mode, it is not possible to do in extension popup view.

If you click X on the request or block the request, the request will never be displayed again until you manually unblock the extension.

chrome://settings/content/handlers

Copy the link manually and open in a new tab.
